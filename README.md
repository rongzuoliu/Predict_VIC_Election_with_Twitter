# Predicting the Results of the Victorian State Elections (2014) with Twitter

## Purpose

A student project for the course of Master of Information Technology in the University of Melbourne. 

## Project Description

Design and Develop an end to end prediction system to forecast the results of the Victorian State Election (2014) by analysing over 100,000 topic-relevant tweets filtered from nearly 10,000,000.

This project includes four stages: tweets harvesting; tweets analysis; results integration, and results visualization by a web front end.
## Content

The source contain only the tweets analysis, results integration, as well as the web front end parts.
